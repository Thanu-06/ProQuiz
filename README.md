# ProQuiz – Programming Quiz App

**ProQuiz** is a **web-based interactive programming quiz application** that helps learners test and improve their coding skills. It supports multiple programming languages and topics, providing a fun and engaging way to practice coding.

## Features
- **Languages Supported:** Python, Java, C, JavaScript, HTML, CSS, DSA, and MySQL.
- **Timed Quizzes:** Each question has a countdown timer to challenge users.
- **Score Tracking:** Tracks correct answers and calculates final scores.
- **Dynamic Results Page:** Shows score, percentage, performance message, and badges.
- **Motivational Badges & Emojis:** Encourages learners with fun rewards.
- **Interactive UI:** Responsive dashboard with hover effects and language selection.
- **Splash Screen:** Professional welcome screen with logo.
- **Retake Quiz:** Users can retry quizzes to improve skills.
- **Static Deployment:** Fully front-end, can be deployed on GitHub Pages.

## Technologies Used

- CSS (Responsive design, grid layout, hover effects)
- JavaScript (Quiz logic, timers, localStorage)

## Usage
1. Open `index.html` in a browser.  
2. The splash screen appears for 3 seconds.  
3. Navigate to the dashboard and select a programming language.  
4. Answer each quiz question within the time limit.  
5. View your results, badges, and performance message on the results page.  
6. Click “Retake Quiz” to try again.

## Deployment
You can deploy this project on **GitHub Pages**:  
1. Push the project to a GitHub repository.  
2. Go to **Settings → Pages**, select branch `main` and folder `/ (root)`.  
3. Save and use the provided GitHub Pages URL to access your app online.
